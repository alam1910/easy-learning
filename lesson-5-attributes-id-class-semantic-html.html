üß© What is an HTML Attribute?

An attribute gives extra information to a tag.

Format:

<tag attribute="value">content</tag>

Example:

<p title="This is a tooltip">Hover over me</p>

üÜî id Attribute (Unique)

  • Used to identify one specific element
  • Must be unique (only one per page)
<h1 id="main-title">My Website</h1>

‚ùå Wrong:

<p id="text">Two</p>

üß± Semantic HTML (Meaningful Tags)

Old way (still works but bad practice):

<div>
  <div>Header</div>
  <div>Content</div>
  <div>Footer</div>
</div>

Modern semantic way:

<header>
  <h1>My Website</h1>
</header>

<main>
  <section>
    <h2>About Me</h2>
    <p>I am learning web development.</p>
  </section>

  <section>
    <h2>Projects</h2>
    <p>Coming soon.</p>
  </section>
</main>

<footer>
  <p>¬© 2026 My Website</p>
</footer>

üß† Why semantic HTML matters

  • Better SEO
  • Better accessibility
  • Easier to read & maintain
  • Industry standard

üß© Common Semantic Tags

Tag Purpose
<header> Top of page/section
<nav> Navigation links
<main> Main content
<section> Grouped content
<article> Standalone content
<footer> Bottom info
