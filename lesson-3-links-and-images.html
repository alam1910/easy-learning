üîó Links (Anchor Tag)

Links let users move between pages or websites.

Basic link syntax

<a href="https://www.google.com">Go to Google</a>

Breakdown

  • <a> = anchor tag
  • href = destination (URL)
  • Text between tags = clickable text

üìÑ Link to another page (local file)

If you have another file like about.html:

<a href="about.html">About Me</a>

Both files must be in the same folder.

üÜï Open link in new tab

<a href="https://www.youtube.com" target="_blank">Visit YouTube</a>

Basic image
